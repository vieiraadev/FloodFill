# 🖌️ Flood Fill - Estruturas de Dados

Projeto acadêmico desenvolvido para a disciplina **Resolução de Problemas Estruturados em Computação (PUCPR)**, ministrada pela **Professora Marina de Lara**.  
O objetivo é implementar o algoritmo **Flood Fill** utilizando **estruturas próprias de Pilha (Stack) e Fila (Queue)** em Java.

---

## 🎯 Objetivos
- Implementar **Pilha e Fila encadeadas** sem usar bibliotecas do Java.  
- Demonstrar a diferença entre **LIFO (Stack)** e **FIFO (Queue)**.  
- Criar **animações** mostrando o progresso do algoritmo.  
- Comparar desempenho entre **DFS (Pilha)** e **BFS (Fila)**.  

---

## 🔧 Estruturas Implementadas
- **Stack (Pilha)** → `push()`, `pop()`, `peek()` — **LIFO**  
- **Queue (Fila)** → `enqueue()`, `dequeue()`, `peek()` — **FIFO**  
- **Complexidade:** O(1) para todas as operações  

---

## 📊 Saídas
- `output_stack.png` → Resultado com Pilha  
- `output_queue.png` → Resultado com Fila  
- `animation/frame_XXXX.png` → Frames da animação  

---

## 📚 Conceitos Aplicados
- **POO:** Classes modularizadas e encapsulamento  
- **Estruturas de Dados:** Implementação própria sem Collections  
- **Algoritmos:** Flood Fill (DFS e BFS)  
- **Complexidade:** O(n), onde *n* é o número de pixels preenchidos
